# Internship Project – React  App

## 📌 Project Overview
This project is developed as part of my **internship (stage)**.  
It is a web application built with **React.js** that demonstrates how to fetch data from an API and display it using **pagination**.

The goal of this project is to improve my skills in:
- React fundamentals
- Working with APIs
- State management using React Hooks
- Clean and organized code structure

---

## 🚀 Features
- Fetch data from an external API
- Display data in a paginated list
- Navigate between pages (Next / Previous)
- Responsive and simple UI
- Clean and readable code

---

## 🛠️ Technologies Used
- **React.js**
- **JavaScript (ES6)**
- **HTML5**
- **CSS3**
- **Fetch API**
